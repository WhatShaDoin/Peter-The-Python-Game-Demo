Mini Glossary

( ) Parentheses
Hold things together or pass things into functions. Memory: Peter’s hands—they hold the stuff!
[ ] Square brackets
Make a list or get items by spot number. Memory: a treasure chest you can reach into.
{ } Curly braces
Make a set or dictionary (named boxes). Memory: a toolbox with labeled drawers.
: Colon
Starts a block (like telling a helper what to do next). Memory: a drum beat before action.
return
Give back an answer from a function. Memory: a boomerang—send work out, answer comes back.
print
Say it on the screen. Memory: Peter’s voice.
-1
The last item in a list. Memory: a back door to the chest.
