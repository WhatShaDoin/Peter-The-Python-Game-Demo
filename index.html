How this works

  • Read the tiny story → it tells you what to build.
  • Type the code yourself (that’s how your brain learns!).
  • Press Run ▶ (or Ctrl+Enter) to check.
  • Pass = ⭐. Oops = 🐞 (a clue to fix).
Peter tip: Bugs aren’t bad. They’re little teachers that say “try again here!”.

Mini Glossary

function a named trick Peter can do again and again
argument the stuff you hand to a function
list a treasure chest that holds many things
index the item’s spot number in the chest
() Peter’s “hands” that hold arguments
[] the treasure chest (lists)
-1 the last spot in the chest
return give back an answer from a function
