Lesson 2: Treasure Chest (lists)

Why this?

  • A list is a treasure chest that holds many items. We’ll add a new gem and learn to peek at the first and last spots.

What you’ll type

  • Create a list, append "diamond", then print the first and last items.

New words & symbols

list treasure chest • append put one more item in
[] the chest • [-1] the back of the chest (last item)

Common oopsies

  • Forgetting quotes around diamond.
  • Typing gems[3] instead of gems[-1] (we want the last one).
  • Printing the list but not the two items.
