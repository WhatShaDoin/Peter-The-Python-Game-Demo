Glossary

Every row has: a tiny image (emoji), the word/symbol, a kid-friendly meaning, and a memory trick. Tap 🔊 to hear it.

ImgWord / SymbolMeaningWay to remember🔊
💬 (function) print() Say it on the screen (shows text). Peter’s voice.
🏷️ variable A labeled box that stores a value. A sticky name tag on a box.
🎁 value The thing inside the box (number, text, etc.). Gift inside the package.
🔤 type The kind of value (like int, str, bool). Shelf label at a store.
#️⃣ # (comment) A note to humans—Python ignores it. A whisper to your future self.
💬 " " or ' ' (quotes) Make a string (text). Speech bubbles around words.
👐 ( ) (parentheses) Hold things together or pass things into a function. Peter’s hands holding stuff.
🥁 : (colon) Starts a block (tells what comes next). A drum beat before action.
↘️ indentation Spaces at the start of a line that show who is inside a block. Stairs that go into a room.
🔘 . (dot) Go inside an object to use a tool or label. A doorbell to a house of tools.
➕ + (add) Add two numbers. Put blocks together.
➖ - (subtract) Take away. Remove a block.
✖️ * (multiply) Repeat groups or scale up. Magic star that copies.
➗ / (divide) Split into equal parts (decimal allowed). Pizza slices with crumbs.
🧱 // (floor div) Divide and round down to an integer. Two boots stomp the number down.
🍪 % (mod) The leftover after dividing. Cookie crumbs that remain.
🚀 ** (power) Raise to a power (3**2 = 9). Double star = rocket power.
📌 = (assign) Put a value into a variable. Pin the label on the box.
🟰 == (equal?) Ask: are they the same? Two equal twins.
🚫 != (not equal) Ask: are they different? No-match sign.
🔼 < > <= >= Compare sizes (smaller / bigger). Crocodile mouth eats the bigger.
➕= += -= *= /= Do math and assign in one step. Shortcut stickers.
🤝 and True only if both are true. Two friends agree.
🔀 or True if at least one is true. Choose this or that.
🔁 not Flips true to false (and back). Opposite day.
✅ True / False / None Yes / No / Empty placeholder. Check, X, and an empty box.
🧵 string (str) A line of text characters. A string of beads (letters).
🪄 f"…" A magic string that shows values inside { }. Curly doors show what’s inside.
🎯 index / slice Pick a spot [i] or a piece [start:stop]. Pointer finger and a cut line.
📏 len() How many items/letters. Ruler that counts.
🧩 string methods .lower(), .upper(), .split(), .join(), .replace()… Little tools on the string’s keychain.
📦 list [ ] An ordered collection you can change. A treasure chest you reach into.
🗂️ tuple ( , ) An ordered collection you can’t change. A photo album—no edits.
🧰 dict { } Labeled drawers: key → value. Toolbox with named slots.
🫧 set { } A bag of unique items (no duplicates). Soap bubbles—no two stick.
🛠️ list methods .append(), .extend(), .insert(), .pop(), .remove()… Chest tools: add/take gems.
🛤️ if / elif / else Choose a path based on a test. Train tracks that split.
🔁 for Do something for every item/number. Marching in a line.
🌀 while Repeat while a test is true. Spiral that stops when told.
⏹️ break / continue / pass Stop loop / skip to next / do nothing. Stop, hop, hands-off.
📚 range() Make counting numbers for loops. Number line in a box.
🧷 enumerate(), zip() Count items or pair lists together. Number pins & zipper merge.
🪄 def Start a function (a named trick). Magic word to teach a trick.
🪃 return Send back an answer from a function. Boomerang comes back.
📨 parameters / arguments The info a function needs (with defaults/keywords sometimes). Mail you hand to the helper.
📖 docstring A help note at the top of a function (triple quotes). Sticky note inside the trick book.
📦 import / from / as Bring tools from other boxes (modules). Borrow your friend’s toolbox.
🏗️ class / object A blueprint and the thing built from it. Recipe and the cake.
🧰 method / attribute Tools and labels inside an object (via dot). Tools in a toolbox drawer.
🧯 try / except / finally Try code, catch errors, clean up. Safety net & broom.
🚨 raise Make an error on purpose. Pull the alarm when needed.
🏷️ lambda A tiny one-line function without a name. Pocket-sized trick.
⚙️ comprehension Quick way to build lists/dicts/sets in one line. Factory belt for collections.
📂 with open(...) Safely open a file and auto-close it. Door that shuts itself.
🪪 is Are these the exact same object? ID card check.
📥 in / not in Is an item inside a collection? Peek inside the chest.
